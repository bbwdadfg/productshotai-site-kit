Shortcodes for linking WordPress content to ProductShot AI workbench and public pages.

== Description ==

Baiwei Content Links for ProductShot AI adds a small shortcode for publishers who mention ProductShot AI in WordPress content.

This plugin is published by Baiwei Chu and uses ProductShot AI in the plugin name only to describe the destination service.

Use `[productshotai_link]` to link to the ProductShot AI workbench, or pass a supported page:

`[productshotai_link page="blog" text="Read ProductShot AI guides"]`

Supported pages are `home`, `workbench`, `pricing`, `blog`, `contact`, and `zh_home`.

Links:

* Website: https://productshotai.app
* Workbench: https://productshotai.app/#workbench
* Source: https://github.com/bbwdadfg/productshotai-site-kit

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/baiwei-content-links-productshot-ai/`.
2. Activate the plugin through the WordPress Plugins screen.
3. Add `[productshotai_link]` to a post, page, or reusable block.

== Frequently Asked Questions ==

= Does this call ProductShot AI APIs? =

No. It only builds public ProductShot AI links for WordPress content.
